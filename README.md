# Sistema de Gerenciamento de Tarefas

Este é um app Django para gerenciamento de tarefas com autenticação de usuários e interface moderna usando Bootstrap.

## Setup

1. Ative o ambiente virtual: `source venv/bin/activate`
2. Instale dependências: `pip install -r requirements.txt`
3. Aplique migrações: `python manage.py migrate`
4. Crie superusuário: `python manage.py createsuperuser`
5. Rode o servidor: `python manage.py runserver`

## Funcionalidades
- Registro e login de usuários
- CRUD de tarefas por usuário
- Interface responsiva com Bootstrap

Consulte a documentação do Django para mais detalhes.
